# Xivex – Fashion E-Commerce Platform

แพลตฟอร์มอีคอมเมิร์ซร้านขายเสื้อผ้าออนไลน์ รองรับการใช้งานอย่างเต็มรูปแบบ พัฒนาด้วย Native PHP และ MySQL เน้นความปลอดภัยและประสิทธิภาพ

## ✨ ฟีเจอร์หลัก
* **สำหรับลูกค้า:** เลือกซื้อสินค้า, ตะกร้าสินค้า, แนบสลิปชำระเงิน, เปลี่ยนภาษา (TH/EN), และระบบลืมรหัสผ่าน
* **สำหรับแอดมิน:** จัดการสินค้า, อนุมัติคำสั่งซื้อ, จัดการสมาชิก, และระบบ Broadcast ข้อความแจ้งเตือนโปรโมชั่น
* **ความปลอดภัย:** ป้องกัน SQL Injection (PDO), CSRF Token, และเข้ารหัสรหัสผ่าน (Bcrypt) อย่างแน่นหนา

## �️ โครงสร้างเทคโนโลยี
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS / ไม่ใช้ Framework)
- **Backend:** PHP 8+
- **Database:** MySQL / MariaDB (XAMPP)

## � วิธีการติดตั้ง
1. ตั้งค่าฐานข้อมูล: สร้าง DB `xivex_store` บน PhpMyAdmin แล้ว Import ไฟล์ที่เกี่ยวข้อง (หรือใช้งานตามที่มี)
2. แก้ไขการเชื่อมต่อ DB ในไฟล์ `includes/db.php` หรือ `includes/config.php` ตามเซิฟเวอร์ของคุณ
3. นำโฟลเดอร์โปรเจกต์ใส่ใน `htdocs` (สำหรับ XAMPP) แล้วเปิดเบราว์เซอร์ลุยได้เลย!

**Login แอดมินทดสอบ (หากมี):**
- Username: `admin` หรือ `admin@xivex.com`
- *อย่าลืมเปลี่ยนรหัสผ่านบนเว็บจริง*
